# ナレッジベースの確認・作成

`knowledge_base.json` はゲームマスターのハルシネーション回避のためのファイル。  
問題追加時のステップ5として、`answer` に登録対象のワードが含まれるか確認し、必要に応じてエントリを追加・更新する。

---

## 登録対象の判定

以下のいずれかに該当するワードが `answer` に含まれる場合、KB エントリを作成する。  
該当しない場合はこのステップをスキップする。

- **慣用句・定型フレーズ**（「指切りげんまん」「七転び八起き」「一石二鳥」など）
- **文化的・宗教的な慣習・行事**（「お盆」「節分」「初詣」など）
- **特定の神話・物語・ゲーム・都市伝説のキャラクターや設定**（「浦島太郎」「桃太郎」など）

---

## 各エントリに含める facts

以下の4観点を必ず調べ、ハルシネーション回避に有効な記述を `facts` に追加する。  
**明確に「〜しない」「〜ではない」と否定する形**で書くことで、誤った回答を防ぎやすくなる。

| 観点 | 調べること | facts の書き方例 |
|---|---|---|
| **登場する生物・人物** | 動物・植物・人物が登場するか。登場する場合、種類・数・属性（性別・年齢・職業など）は何か | 「動物は登場しない」「人間のみ登場する」「〜という種類の動物が登場する」 |
| **時代** | 現代の話か、過去・未来の話か | 「現代の慣習である」「江戸時代に由来する」 |
| **現実/架空** | 現実に存在する慣習・事実か。物語・神話・ゲーム・夢・都市伝説などの架空の話か | 「現実の慣習である」「民話・昔話の登場人物である」 |
| **地域・場所** | 日本の話か、海外の特定の国・地域か。地球外・海中など特別な場所があるか | 「日本固有の慣習である」「特定の地域に限定されない」 |
| **職業・犯罪** | 特定の職業（医者・弁護士・聖職者など）や犯罪（窃盗・殺人など）が深く関わるか | 「特定の職業とは無関係である」「〜という職業が深く関係する」「犯罪行為は含まれない」 |

---

## 既存エントリの更新

`knowledge_base.json` に同一キーワードがすでに存在する場合は、新規作成ではなく **facts の追記・修正** を行う。

---

## 確認表示

`knowledge_base.json` への書き込み前に以下の形式で表示し、確認を求める：

```
【ナレッジベース追加候補】

キーワード: <ワード>
description: <一般的な定義（1〜2文）>
facts:
- <fact 1>
- <fact 2>
...

knowledge_base.json に追加してよいですか？（既存エントリの場合は「更新」と表示）
```

---

## knowledge_base.json への追加・更新

確認が取れたら以下の形式で `knowledge_base.json` に書き込む：

```json
{
  "<キーワード>": {
    "description": "<一般的な定義>",
    "facts": [
      "<fact 1>",
      "<fact 2>"
    ]
  }
}
```
